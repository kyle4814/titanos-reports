A passive security assessment of soliddocuments.com (206.82.203.59) revealed 11 findings across 7 open ports and 0 subdomains. Overall risk grade: C.

Findings

HIGH Port 21/tcp (FTP) Open

MEDIUM HTTP Does Not Redirect to HTTPS

Remediation: Add HTTP→HTTPS redirect (301) in web server config.
MEDIUM Missing CSP
Missing CSP — vulnerable to XSS injection
Remediation: Add 'Content-Security-Policy' header to server response.
MEDIUM Port 25/tcp (SMTP) Open
SMTP exposed — potential open relay, spam abuse. Detected service: Exim smtpd 4.95.

LOW Missing X-Content-Type-Options

LOW Missing X-XSS-Protection
