Executive Summary

A passive security assessment of dhub.org (35.209.110.136) revealed 17 findings across 14 open ports and 9 subdomains. Overall risk grade: F.

2 critical vulnerabilities were identified that require immediate attention. These represent high-risk exposure points that could allow unauthorized access or data breach. 4 high-severity issues were found that should be addressed within the next 7 days. 8 medium-severity findings were noted for near-term remediation. 3 low-severity items represent best-practice improvements.

Findings

CRITICAL Port 3306/tcp (MySQL) Open
MySQL exposed — database directly accessible. Detected service: MySQL.
Remediation: Close port 3306 or restrict access to trusted IPs via firewall. If required, ensure MySQL is patched and properly authenticated.
CRITICAL Port 5432/tcp (PostgreSQL) Open
PostgreSQL exposed — database directly accessible. Detected service: PostgreSQL DB 9.6.0 or later.
Remediation: Close port 5432 or restrict access to trusted IPs via firewall. If required, ensure PostgreSQL is patched and properly authenticated.
